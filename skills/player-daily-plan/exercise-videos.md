## How to Add a Video

1. Upload the recording to Cloudinary via the app (Indstillinger → Upload video).
2. Copy the secure URL (starts with `https://res.cloudinary.com/...`).
3. Paste it in the URL column next to the exercise name above.
4. The next generated plan that includes that exercise will automatically link to the video.
