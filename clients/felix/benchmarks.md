# Benchmarks — Felix

Hard numbers we can point back to. Append-only log — every test session adds a new entry with the date, never overwrite an old row. Re-test cadence: **every 6–8 weeks**, and before/after each tournament.

These numbers anchor the `working_score` values in [profile.md §3](profile.md) and the progress evidence in [history.md](history.md).

---

## Metrics to collect

| Metric | How to test | Why it matters |
|---|---|---|
| 10m sprint (no ball) | Standing start, stopwatch, best of 3 | Pure acceleration baseline |
| 10m dribble (with ball) | Same start/finish, ball at feet the whole way | `dribbling_speed` — the metric that cornered his current ceiling |
| Sprint-vs-dribble delta | 10m dribble minus 10m sprint | How much the ball slows him down — the gap we're closing |
| Juggling — right foot only | Max count, 3 attempts | `ball_mastery` + dominant-foot baseline |
| Juggling — left foot only | Max count, 3 attempts | `weak_foot` |
| Juggling — alternating | Max count, 3 attempts | Coordination under control |
| Wall passes in 30s (right) | Stand 3m from wall, 1-touch returns only | `passing` + `first_touch` combined |
| Wall passes in 30s (left) | Same, weak foot | `weak_foot` applied |
| T-drill time | Standard 10-10-10 T-drill | `agility` |
| Shots on target / 10 from 12m | Stationary ball, no keeper | `finishing` baseline |
