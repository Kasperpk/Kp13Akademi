# Clients

Individual player management. One folder per client.

## How to Add a New Client

1. Copy the `_template/` folder
2. Rename it to the player's name (e.g., `lucas/` or `lucas-hansen/`)
3. Fill in `profile.md` and `goals.md`
4. Add a session file after each training in `sessions/`
5. Keep running observations in `notes/ongoing.md`

## Structure per Client

```
clients/
└── player-name/
    ├── profile.md        — who they are
    ├── goals.md          — what they're working toward
    ├── sessions/         — one file per training session
    │   └── YYYY-MM-DD.md
    └── notes/
        └── ongoing.md    — running coaching observations
```
