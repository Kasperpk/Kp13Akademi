# Team — U9

U9 group sessions. Two trainings per week, ~90 minutes each.

## Structure

### `sessions/`
Individual session files — both generated (via the generator tool) and manually planned.

Includes:
- Sessions generated by `../generator/` (output lands here automatically)
- Standalone sessions (indoor, strength, etc.)

### `curriculum/`
Structured course content (proforløb). Pre-planned session sequences organized by level.

- `proforløb1/` — Foundation level (sessions 1-10)
- `proforløb2/` — Advanced level

## Generating a New Session

From the `generator/` directory:

```bash
cd ../generator
python generate.py
```

Select the `u9_team` template. The output is saved to `team/sessions/`.

## Session Format

Each session follows the standard structure:

1. **Opvarmning** (Warm-up) — 10 min
2. **Rondo** — 10-15 min
3. **Teknisk blok / Positionsspil** — 15-20 min
4. **Spilform** (Game form) — 20-25 min
5. **Frit spil** (Free play) — 10-15 min
6. **Afslutning** (Cool down) — 5 min
