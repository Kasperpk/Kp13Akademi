# Training Methodology — Evidence Base

The research-backed reference for designing KP13 academy sessions and the app's
home-training. Use this as the quality bar: every activity should be defensible
against the principles below. See `VISION.md` for the product, and the three
coaching pillars (Technical / Tactical / Mentality) for the KP13 style.

Scope: most relevant to ages 8–12 (U9–U12). Last reviewed: 2026-05-15.

## Core principles (what the evidence converges on)

1. **Maximize touches.** The binding constraint at U9–U12 is time on the ball.
   Small numbers, small fields, no queues. Design so every player gets as many
   touches as possible.

2. **Game-realistic over isolated drills (constraints-led approach).**
   Isolated drills and constraints-led games produce *similar* technical gains,
   but constraints-led games are clearly superior for decision-making,
   perception, and tactical transfer to matches. Prefer constraining a game
   ("goal counts after 3 players touch it") over running a static drill.

3. **Coerver ball mastery — as activation, not the whole session.**
   Ball mastery and 1v1 moves build the "comfortable on the ball" base. The
   documented limit: pure repetition without opponents does not transfer. Use
   ~10–15 min, then put the skill under pressure in games.

4. **Differential learning — build in variability.** Don't groove one "correct"
   technique. Expose players to constantly varied versions (different surfaces,
   off-balance, varied speeds). Evidence shows gains in technical, creative, and
   positional performance. Practically: never repeat a rep identically.

5. **Guided discovery (Play–Practice–Play).** Start with a game, briefly
   practice what broke down, return to the game. Coach by asking questions and
   adjusting constraints — not lecturing. Keep coach talk under ~20 seconds.

6. **Fun is a performance variable.** In a session full of laughter, players
   stop fearing mistakes and learn faster. Emotional/social learning matters as
   much as technical work at this age.

## Session structure (U9, ~80–90 min)

| Block | Time | Purpose |
|---|---|---|
| Dynamic warm-up / ball mastery | 15 min | Touches + Coerver base, with variability |
| 1v1 station | 15 min | Core duel, parallel games = no waiting |
| Constraint game (technical focus) | 15 min | Train a skill without an isolated drill |
| Free small-sided game | ~25 min | Decision-making, transfer, joy |
| Cool-down | 5 min | Social/emotional close |

## Practical rules for U9–U12

- Small fields (~25×30m), small goals, no fixed goalkeeper.
- No heading at this age. Shinguards on, no jewellery.
- Fix failing skills by changing a constraint, not by lecturing.
- Reps should never look identical (differential learning).
